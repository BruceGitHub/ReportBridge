# Adoption Metrics

Keep metrics simple and actionable.

## Suggested KPIs

- rb-updated-docs-per-week
- onboarding-time-to-first-merged-pr
- repeated-incident-rate
- unresolved-known-issues-count
- adr-coverage-for-major-decisions

## Review Cadence

- weekly: operational indicators
- monthly: trend and quality review

## Interpretation Notes

- more docs is not always better; focus on usefulness
- a drop in repeated incidents is a strong positive signal
- if onboarding time grows, improve guides and standards first
