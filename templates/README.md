# Templates

Template library organized by category.

## Categories

- architecture: ADR and architectural decision support
- incidents: bugs and incident runbooks
- team: onboarding and team operational docs
- quality: coding conventions and quality baselines
- metrics: adoption and process metrics
- procedures: customer-facing operational procedures
- knowledge: FAQ, glossary, mission, and shared knowledge artifacts
- configuration: settings, flags, and configuration explanations
